Template Installation

  1. FTP Upload:
    • Open up your FTP manager and connect to your hosting
    • Browse to required directory (Normally public_html)
    • Upload the files inside foran folder.

Pages

    • index.html - is for make a beutiful homepage.
    • index-2.html - is for make a beutiful homepage.
    • about-us.html - is for about page.
    • services.html - is for services page.
    • portfolio.html - is for create a beautiful portfolio page
    • portfolio-nomargin.html - is for create a beautiful portfolio page with no margin portfolio.
    • portfolio-fullwidth.html - is for create a beautiful portfolio page with fullwidth portfolio.
    • portfolio-nomargin-fullwidth.html - is for create a beautiful portfolio page with no margin & fullwidth portfolio.
    • portfolio-details.html - is for create a beautiful portfolio details page.
    • blog.html - is for blogs page.
    • blog-left-sidebar.html - is for blogs page with left sidebar.
    • blog-right-sidebar.html - is for blogs page with right sidebar.
    • blog-details.html - is for blog details page.
    • blog-details-leftsidebar.html - is for blog details with left sidear page.
    • blog-details-rightsidebar.html - is for blog details with right sidear page.
    • contact-us.html - is for contact page.
    • faq.html - is for frequently ask question page.
    • 404.html - is for 404 page.
